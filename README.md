# Digital Healthcare Management System (Backend)

Backend for a multi-role hospital system:

- Patient
- Doctor
- Receptionist
- Medical (Pharmacy)
- Radiology

Built with:

- Flask
- PostgreSQL
- SQLAlchemy
- JWT Auth
- Argon2 password hashing
- ReportLab PDF prescriptions
- Docker & Docker Compose
